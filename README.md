# AI Smart Ingredient Scanner

An intelligent kitchen assistant that uses **Computer Vision** to recognize food ingredients and suggest recipes in real-time. Built with Python and Google's Gemini AI.



## How It Works
1. **Snap a Photo:** Upload a picture of your fridge, pantry, or specific ingredients.
2. **AI Analysis:** The app uses a multimodal Large Language Model (LLM) to identify every item in the frame.
3. **Chef Mode:** The AI acts as a Michelin-star chef to suggest creative recipes based *only* on the items it sees.

## Tech Stack
* **Frontend:** [Streamlit](https://streamlit.io/) (Web Interface)
* **AI Brain:** [Google Gemini API](https://aistudio.google.com/) (Vision & NLP)
* **Language:** Python 3.10+
